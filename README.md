# 🚀 Praticando Automação com Playwright & JavaScript

Este projeto tem como foco o aprendizado e a aplicação de boas práticas em **Engenharia de Qualidade de Software**. Através deste repositório, busco consolidar o uso do framework **Playwright** para automação de testes de alta performance.


# 🎯 Objetivo do Projeto

Demonstrar a implementação de uma suíte de testes robusta utilizando o padrão **Page Object Model (POM)**, cobrindo fluxos críticos de um e-commerce (SauceDemo).
---

## 🚀 Tecnologias Utilizadas

* **Framework:** [Playwright](https://playwright.dev/)
* **Linguagem:** JavaScript
* **Padrão de Projeto:** Page Object Model (POM)
* **Relatórios:** Playwright HTML Reporter

---

## 📋 Cenários de Teste Automatizados

### End-to-End (E2E)
- [x] **Login:** Autenticação de usuário com credenciais válidas.
- [ ] **Fluxo de Compra:** Adicionar produto ao carrinho e validar a transição de páginas.
- [ ] **Checkout Completo:** Finalização da compra com preenchimento de dados e validação de sucesso. (Em desenvolvimento)

### API
- [ ] **Validação de Endpoints:** Testes de contrato e status code. (Planejado)
